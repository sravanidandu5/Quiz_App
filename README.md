Quiz Application
A simple web-based quiz application built using HTML, CSS, and JavaScript.
It includes a Welcome page, Login page, Question page, and Result page with score display.

Features
Username & password login
15 quiz questions
Next button to move question-by-question
Submit button on last question
Final score with “Thank you / All the best” message
Clean UI with custom CSS

Technologies Used
HTML5
CSS3
JavaScript

How to Run
Download or clone the repository
Open index.html in any browser
Proceed through login → quiz → result
